官方解释闭包就是指有权访问另一个函数作用域中的变量的函数.
创建闭包常见方式,就是在一个函数内部创建另一个函数.
```bash
function example() {  
    var a = 12;  
    function bibao() {    
        console.log(a);     //12  
    }
}
```
在上述例子中在example函数中a是局部变量，一般函数外是访问不到的。但是在函数内bibao这个函数却能访问到a这个变量，这就是闭包，优点在于不至于内存外泄，实现封装的功能。但是这样会消耗巨大的存储空间，因此闭包的使用还是慎重一点比较好。
